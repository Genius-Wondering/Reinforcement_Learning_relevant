# Polynomial Basis（多项式基函数逼近）
---
# 1. Motivation（为什么提出）

**历史背景**：多项式逼近是函数逼近理论中最古老的方法，可追溯到 Taylor（泰勒展开）与 Weierstrass（1885年证明连续函数可被多项式一致逼近）。它是几乎所有后续逼近理论（Fourier、Wavelet、样条）的思想起点。

**解决什么问题**：在没有解析表达式、只有采样点或函数值的情况下，如何用一个简单、可计算、可微的函数去近似复杂函数？

**为什么以前的方法不好**：在多项式之前，逼近手段主要依赖插值表或分段线性近似，精度低且难以求导、难以做进一步的分析（如求极值、积分）。多项式提供了光滑、可解析处理、任意阶可导的表示方式。

---
# 2. Mathematical Idea（数学思想）

**一句话解释**：用简单幂函数 1, x, x², x³, ... 的线性组合去逼近任意光滑函数。

**几何解释**：在函数空间中，{1, x, x², ...} 张成一个无穷维子空间；逼近就是把目标函数投影（或截断展开）到这个子空间的某个有限维子空间上。

**数学解释**：任意光滑函数在某点附近可以用其各阶导数信息（Taylor 展开）重构；多项式基提供了这种局部信息的全局表示形式。

---
# 3. Mathematical Definition（数学定义）

给定阶数 n，多项式逼近定义为：

```
f̂(x) = w₀ + w₁x + w₂x² + ... + wₙxⁿ = Σᵢ₌₀ⁿ wᵢxⁱ
```

**符号说明**：
- x：输入变量（可以是标量或向量，向量情形需要多元多项式，如 x₁ᵃx₂ᵇ）
- wᵢ：待学习的系数（参数）
- n：多项式阶数（决定模型复杂度）

**变量解释**：n 越大，函数类越丰富，逼近误差越小，但估计误差（过拟合风险）越大 —— 这正是第零章"逼近误差分解"的直接体现。

---
# 4. Basis Function（基函数）

**为什么这样设计**：幂函数集合 {1, x, x², ..., xⁿ} 是最直观的一组基，构造简单、求导积分封闭（导数/积分仍是多项式）。

**Basis长什么样**：φᵢ(x) = xⁱ，i = 0, 1, ..., n

**举例**：n=3 时基函数为 {1, x, x², x³}

**画图（文字描述）**：φ₀=1 是常数直线；φ₁=x 是过原点的斜线；φ₂=x² 是抛物线；φ₃=x³ 是S形曲线。阶数越高，基函数在定义域两端的振荡幅度越大（这是 Runge 现象的根源）。

---
# 5. Approximation Formula（逼近公式）

统一写成：

```
f(x) ≈ Σᵢ wᵢφᵢ(x)，其中 φᵢ(x) = xⁱ
```

**如何得到**：可以通过 Taylor 展开在某点附近截断得到，也可以通过最小二乘拟合采样点得到。

**为什么是线性的**：虽然基函数 φᵢ(x)=xⁱ 关于 x 是非线性的，但 f̂ 关于参数 w 是线性的（w 只做线性加权求和），因此优化问题是凸的、有闭式解——这是"线性基展开"这一大类方法共同的核心特征。

---
# 6. Mathematical Foundation（数学基础）

- **Taylor 展开**：局部多项式逼近的理论来源，n 阶 Taylor 多项式在展开点附近误差为 O(xⁿ⁺¹)
- **Weierstrass 逼近定理**：闭区间上任意连续函数都可以被多项式一致逼近到任意精度（存在性保证，但未给出如何构造）
- **Approximation Theory**：Jackson 定理给出了逼近误差与函数光滑度、多项式阶数之间的定量关系

---
# 7. Approximation Ability（逼近能力）

**为什么能逼近任意函数**：由 Stone-Weierstrass 定理保证，在紧集上多项式函数类在连续函数空间（配合上确界范数）中稠密。

**误差如何分析**：若 f 是 k 次连续可微，则 n 阶多项式逼近误差满足 ||f − f̂||∞ = O(n⁻ᵏ)（光滑度越高收敛越快）。

**是否谱收敛**：只有 f 是解析函数时才有指数收敛（谱收敛）；一般光滑函数只有代数阶收敛。

**是否局部收敛**：否——多项式是**全局基**，改变一个系数会影响整个定义域上的逼近，没有局部性。

**是否一致收敛**：等距节点插值高阶多项式会出现 **Runge 现象**（边界振荡发散），需要用 Chebyshev 节点或最小二乘拟合而非插值来避免。

---
# 8. Optimization（参数学习）

- **Least Squares**：给定采样点 {(xⱼ, yⱼ)}，最小化 Σⱼ(yⱼ − f̂(xⱼ))²，由于关于 w 线性，有闭式解 w = (ΦᵀΦ)⁻¹Φᵀy
- **Gradient Descent**：数据量大时用 SGD 迭代更新 w，避免矩阵求逆
- **TD Learning**：在 RL 中，用 TD 误差 δ = r + γf̂(s') − f̂(s) 替代监督误差，做半梯度更新 w ← w + αδ∇ᵥf̂(s)
- **LSTD**：直接求解投影贝尔曼方程的闭式解，避免逐步迭代，样本利用率更高

---
# 9. RL中的应用

- **表示V**：V̂(s;w) = Σᵢ wᵢsⁱ（s 为标量特征时）
- **表示Q**：Q̂(s,a;w) = Σᵢ wᵢφᵢ(s,a)，需要构造联合特征（通常做多元多项式扩展）
- **表示Policy**：较少直接用多项式表示策略，因为策略需要归一化为概率分布，多项式没有天然的非负性约束
- **对应算法**：Linear TD、Linear SARSA、Linear Q-learning，早期文献中常作为基线方法

---
# 10. Theoretical Properties（理论性质）

- **是否线性**：关于参数线性，关于输入非线性
- **是否凸优化**：是（监督学习设定下为凸二次问题）
- **是否保证收敛**：在 on-policy TD 下有收敛性证明（Tsitsiklis & Van Roy, 1997）；off-policy 下可能发散（Deadly Triad）
- **是否稳定**：阶数高时数值不稳定（Vandermonde 矩阵条件数随阶数指数增长）
- **复杂度**：参数量 O(n)（一维），高维输入下组合爆炸（O(nᵈ)，d为输入维度）

---
# 11. Advantages（优点）

- 数学形式简单，求导、积分解析封闭
- 参数少时计算高效，闭式解存在
- 理论成熟，收敛性分析完整

---
# 12. Disadvantages（缺点）

- 全局基函数，局部特性差，某处误差会污染全局
- 高阶多项式在边界振荡（Runge 现象），数值不稳定
- 维度灾难：多元多项式基数量随维度指数增长，不适合高维状态空间

---
# 13. Comparison（与其他方法比较）

| 方法 | 局部性 | 光滑函数收敛速度 | 高维可扩展性 | 数值稳定性 |
|---|---|---|---|---|
| Polynomial | 差（全局基） | 代数阶 | 差 | 阶数高时差 |
| Fourier | 差（全局基，但正交） | 谱收敛（解析函数） | 中等 | 好 |
| Wavelet | 好（局部支撑） | 依小波光滑度 | 好 | 好 |
| RBF | 好（局部支撑） | 依带宽选择 | 中等 | 中心选择敏感 |
| Tile Coding | 好（局部） | 低（分段常数级） | 好，计算快 | 好 |
| MLP | 自适应（数据驱动） | 依赖数据/宽度 | 好 | 训练依赖初始化 |

---
# 14. Applications（实际应用）

- 早期 **Mountain Car**、**CartPole** 等经典控制任务的基线特征表示
- 经济学/控制理论中低维状态的价值函数拟合
- 现代 RL 中较少单独使用，多作为教学案例或与其他基函数组合使用
